Portfolio BTS SIO

📋 Description du projet

Ce portfolio a été développé dans le cadre de ma formation BTS SIO. Il présente mes compétences, mes projets et mon parcours de manière interactive et moderne. Le site intègre des effets de parallaxe, un système de thème clair/sombre, et des animations fluides pour offrir une expérience utilisateur optimale.

🎯 Objectifs

  • Présenter mon parcours et mes compétences de manière professionnelle
  • Démontrer mes capacités en développement front-end
  • Créer une interface moderne et responsive
  • Implémenter des animations et effets visuels

💻 Technologies utilisées

HTML5 CSS3 JavaScript ES6+ Tailwind CSS

✨ Fonctionnalités principales

  • Effets de parallaxe sur les sections
  • Système de thème clair/sombre avec sauvegarde
  • Graphique de compétences interactif avec bulles
  • Carousel de projets avec navigation
  • Design responsive (mobile, tablette, desktop)
  • Animations fluides et transitions

Script d'Analyse de Données

📋 Description du projet

Collection de scripts Python pour l'automatisation de l'analyse de données. Les scripts permettent de traiter des fichiers CSV, d'effectuer des analyses statistiques, de générer des visualisations et de produire des rapports automatiquement.

🎯 Objectifs

  • Automatiser le traitement de grandes quantités de données
  • Créer des visualisations pertinentes et informatives
  • Générer des rapports d'analyse automatiquement
  • Optimiser les performances de traitement

💻 Technologies utilisées

Python 3.10+ Pandas Matplotlib NumPy

✨ Fonctionnalités principales

  • Import et nettoyage de données CSV/Excel
  • Analyse statistique descriptive
  • Génération de graphiques (lignes, barres, scatter, etc.)
  • Détection d'anomalies et de tendances
  • Export des résultats en PDF
  • Logging et gestion des erreurs
